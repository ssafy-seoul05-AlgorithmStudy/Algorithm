## 파일 업로드 규칙
### 폴더 명 :file_folder:
### 'BOJ' or 'SWEA'+ '문제 번호'
문제 유형 폴더 하위 폴더로 **문제별 폴더 생성 후** 폴더에 소스파일 업로드 할 것 <br>
ex) 2.MST/BOJ1234/BOJ1234_hyunjin.java

<br>

### 파일 명 :page_with_curl:
### 'BOJ' or 'SWEA'+ '문제 번호_이름'
- 파일명 "`BOJ1234_hyunjin`"처럼  **양식 통일**🙏🏻
